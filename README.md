# 💰 Simple Interest Calculator

## 📌 Project Description

This project is a **Simple Interest Calculator** that helps users calculate interest based on principal amount, rate of interest, and time period. It is a basic project designed to understand fundamental programming and mathematical concepts.

---

## 🧮 Formula Used

Simple Interest is calculated using the formula:

[
\text{Simple Interest} = \frac{P \times R \times T}{100}
]

Where:

* **P** = Principal Amount
* **R** = Rate of Interest
* **T** = Time (in years)

## 📷 Example

Enter:

* Principal = 1000
* Rate = 5%
* Time = 2 years

Output:

* Simple Interest = 100
