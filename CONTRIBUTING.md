# 🤝 Contributing Guidelines

Thank you for your interest in contributing to this project! 🎉

We welcome all types of contributions to improve this Simple Interest Calculator.

---

## 🚀 Ways to Contribute

### 🐞 Bug Reports

* Report bugs by creating an issue
* Clearly describe the problem
* Include steps to reproduce the issue

---

### 🔧 Bug Fixes

* Fix existing bugs
* Ensure your code works properly
* Test before submitting

---

### 📚 Documentation Improvements

* Improve README.md or other documentation
* Fix grammar or add better explanations

---

### ✨ Enhancements

* Add new features
* Improve UI/UX
* Optimize performance

---

### 💡 Ideas

* Suggest new ideas for improving the project
* Share creative features

---

## 📌 Contribution Steps

1. Fork the repository
2. Create a new branch
3. Make your changes
4. Commit your changes
5. Submit a pull request

---

## ✅ Guidelines

* Write clear and simple code
* Follow project structure
* Be respectful and constructive

---

## 🎯 Conclusion

All contributions, big or small, are welcome and appreciated!
